# HMW — Halande Motoren Werke

HMW (Halande Motoren Werke) is a premium sports car brand concept website designed with an ultra-luxury aesthetic — combining Ferrari-level performance energy with Apple-style minimalism. (This is a totally fictional company)

This project is a cinematic dark-mode landing page featuring:

- High-end typography and whitespace-driven layout  
- Luxury showroom-inspired UI design  
- Premium hero section built for future animation  
- Flagship car model showcase (V8, V12, W16 lineup)  
- Testimonials + performance review styling  
- Concierge-style call-to-action flow  
- EmailJS-powered confirmation email templates  
- Fully responsive design (mobile → desktop)

Built for modern deployment and fast performance.

### Tech Stack
- HTML5 + CSS3 (Flexbox/Grid)
- Minimal JavaScript interactions
- Netlify-ready production structure
